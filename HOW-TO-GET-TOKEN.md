# HOW TO GET TOKEN

`https://developers.facebook.com/tools/explorer`

1. Chọn ứng dụng của bạn ở góc trên bên phải.
2. Chọn quyền `pages_manage_posts` và `pages_read_engagement` trong phần "User or Page" ở bên trái.
3. Người dùng hoặc Trang: Chọn "Get User Access Token"
4. Nhấn nút "Generate Access Token" để lấy token.
